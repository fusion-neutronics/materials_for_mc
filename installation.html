Installation#

materials_for_mc can be installed and used on Linux, Windows, and macOS.

Python Installation#

materials_for_mc is available on PyPI. To install, run:

pip install materials-for-mc

Rust Installation#

materials_for_mc is published as a Rust crate. To add it to your project, include the following in your Cargo.toml:

[dependencies]
materials_for_mc = "*"

Then run:

cargo build
